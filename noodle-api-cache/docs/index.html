Noodle Api Proxy

Cache any API request to prevent exceeding your API limit!

Usage:

Simple append the following URL to your request URL

https://noodle-api-cache.herokuapp.com/?expiry=86400&url=

Example:

https://thesimpsonsquoteapi.glitch.me/quotes?count=10

Becomes

https://noodle-api-cache.herokuapp.com/?expiry=1000&url=https://thesimpsonsquoteapi.glitch.me/quotes?count=10

Advanced:

The expiry = property specified how long the API response should be cached for. It is set in seconds. If you change the value a new API request is made and the expiry updated. The default if no value is 1 day (86400 seconds)
